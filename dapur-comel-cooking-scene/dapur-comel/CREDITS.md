# Asset Credits

Dapur Comel uses open source game assets. Thank you to these projects.

## Sprites & illustrations

### OpenMoji — CC BY-SA 4.0
Colour food illustrations, cat-face expressions for the Oyen mascot, and
several utensils.

- Website: https://openmoji.org
- Source: https://github.com/hfg-gmuend/openmoji
- License: https://creativecommons.org/licenses/by-sa/4.0/
- Files: `public/assets/game/food/*`, `public/assets/game/cat/*`, and the
  colour utensils in `public/assets/game/tools/*` (bowl, cup, knife, pan,
  plate, spoon).

### game-icons.net — CC BY 3.0
Kitchen tool silhouettes (chef toque, cooking pot, whisk, oven, etc.),
recoloured via CSS and used as the chef hat and tool icons.

- Website: https://game-icons.net
- Source: https://github.com/game-icons/icons
- License: https://creativecommons.org/licenses/by/3.0/
- Authors used: Delapouite, Skoll.
- Files: silhouette icons in `public/assets/game/tools/*`.
